Share text/file via QR code.

Installing
----------

### Binaries ###

See [the releases page](https://github.com/ARwMq9b6/qrs/releases)

### Compiling Yourself ###

[Install Rust toolchain](https://www.rust-lang.org/downloads.html) and then

```bash
$ cargo +nightly install qrs
```
